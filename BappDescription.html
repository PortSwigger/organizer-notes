Organizer Notes streamlines penetration testing workflows by providing a comprehensive note-taking system that integrates directly with HTTP requests. The extension allows you to create, categorize, and attach notes to requests sent to the Organizer, ensuring important insights are preserved alongside the associated HTTP traffic.

Features

  • Create and manage notes with customizable categories for organized documentation
  • Right-click context menu integration to attach notes directly to HTTP requests
  • Automatic note attachment when sending requests to the Organizer with annotations
  • Full CRUD operations for both notes and categories through a dedicated "OrgNotes" tab
  • Import and export functionality for notes in JSON format for backup and sharing
  • Support for uncategorized notes alongside categorized organization
  • Search and selection interface for quickly finding and applying relevant notes

Usage

  1. Access the "OrgNotes" tab to create and manage your notes and categories.
  2. Add categories using the Manage Category button to organize your notes by vulnerability type, testing phase, or any classification system.
  3. Create notes by clicking Add Notes, entering your content, and selecting the appropriate category.
  4. Right-click on any HTTP request in Proxy, Target, or other tools to access the context menu.
  5. Select a note from the categorized menu options to attach it to the request.
  6. The request will be automatically sent to the Organizer with your selected note attached as an annotation.
  7. Use the Import/Export functionality to backup your notes or share them across different Burp installations.
